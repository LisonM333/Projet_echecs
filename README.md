# Partie 3D du projet d'échecs

La branche 3D affiche un plateau de jeu avec les pièces posées dessus. Il comporte une caméra Trackball centrée au milieu de la fenêtre. Il est possible de zoomer avec le scroll de la souris et de regarder autour en cliquant sur le clic droit de la souris. Il y a également une illumination avec le modèle de Phong.

La freefly camera est implémentée dans glimac/FreeflyCamera.hpp | glimac/src/FreeflyCamera.cpp mais n'est pas utilisée.
